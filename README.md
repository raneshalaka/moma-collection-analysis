# MoMA Collection Analysis
A business analysis of the Museum of Modern Art's permanent collection using Python and public data.

## Overview
This project analyzes 160,000+ artworks from MoMA's open-access collection dataset to uncover 
trends in gender representation, geographic diversity, and acquisition patterns across 97 years 
(1929–2026). The goal is to answer a question any museum board would care about: 
**who does MoMA collect, and is that changing?**

## Key Findings
- **Gender:** Female artist acquisitions grew from 2.4% in the 1960s to 28.1% post-2015 — 
  significant progress, but still far from parity
- **Nationality:** American artists account for 51.1% of the collection; French artists are a 
  distant second at 14.6%
- **Acquisition model:** Bulk donations drive collection spikes — a 2008 surge of 5,000+ 
  Drawings & Prints acquisitions traces to two major gifts (Louise Bourgeois and the Fluxus collection)
